#1 🫧 Bubble Blaster
A fast-paced bubble popping game inspired by "Computer Coding for Kids".

#2 🎮 How to Play
* Movement: `W` `A` `S` `D` or Arrow Keys
* Objective: Pop bubbles to score. Every 500 points = Level Up!
* Levels: 30 Levels + 3 Boss Bubbles
* There are many kinds of bubbles when playing find out what happens on your own!
